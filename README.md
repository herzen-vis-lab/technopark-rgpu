# Technopark Child

Дочерняя тема WordPress для сайта Технопарка РГПУ им. А. И. Герцена.

## Стек

- **WordPress**, дочерняя тема от родительской `twentytwentyfour` (подключается через `@import` в [css/style.css](css/style.css)).
- **WindPress** — плагин, генерирующий Tailwind-подобный CSS (`uploads/windpress/data/main.css`), подключаемый первым в цепочке стилей. Вся вёрстка в шаблонах использует его utility-классы (`flex`, `max-w-7xl`, `text-body` и т.д.).
- Кастомные типы записей и метабоксы вместо ACF — вся логика администрирования написана вручную в `functions.php`.
- Swiper — для каруселей (аудитории, команда, галереи).

## Установка

1. Скопировать в `wp-content/themes/`, рядом должна быть установлена родительская тема `twentytwentyfour`.
2. Установить и настроить плагин **WindPress** (генерирует `uploads/windpress/data/main.css`, без него стили не подключатся).
3. Активировать тему в админке — появится пункт меню **«Технопарк»** для управления контентом.

## Кастомные типы записей (CPT)

Все типы регистрируются в `functions.php` и собраны в одном пункте меню админки **«Технопарк»**.

| CPT | Назначение |
|---|---|
| `header_settings` | Логотип и пункты меню шапки (с поддержкой вложенных подменю) |
| `footer_settings`  | Документы, контакты и соцсети (SVG-иконки) в подвале |
| `logo`             | Логотипы партнёров в блоке над hero-секцией главной |
| `hero`             | Заголовок, текст, фото и кнопки hero-секции главной |
| `stats`            | Карточки статистики («Мы в цифрах») |
| `directions`       | Направления деятельности (иконка, галерея, кнопки, краткое описание) |
| `faq`              | Вопросы и ответы для аккордеона |
| `carousel`         | Слайды карусели аудиторий (фото + кнопки) |
| `team`             | Сотрудники (для карусели команды) |
| `document`         | Документы со ссылками (используются в футере/страницах) |
| `about_content`    | Контент страницы «О нас» (заголовок, текст, галерея, гербы) |
| `meropriyatiya`    | Мероприятия (даты, описание, галерея, карточки особенностей, кнопки) |
| `project`          | Коробочные образовательные продукты (КОП), с таксономиями `project_direction` и `project_age` |

Для CPT с медиа (лого, галереи, SVG-иконки) используются повторяющиеся метабоксы на базе `wp.media`; поддерживается загрузка SVG (включена через `upload_mimes`).

## Известные особенности

- Часть код-базы дублируется (метабоксы-загрузчики медиа, повторители кнопок) — кандидаты на рефакторинг в общие функции.
- Нет проверки nonce в обработчиках сохранения метабоксов.
- SVG выводится в шаблоны напрямую (`file_get_contents`) без санитизации — стоит ограничить доступ к загрузке медиа доверенными пользователями.
- `js/script.js` и файлы `css/components/*.css` сейчас не задействованы активно.
